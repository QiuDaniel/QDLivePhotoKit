# QDLivePhotoKit
Generate Live Photo from mp4 file

## Installation

### CocoaPods

The preferred installation method is with [CocoaPods](http://cocoapods.org). Add the following to your Podfile:
```ruby
pod 'QDLivePhotoKit', '~> 0.1.0'
```
